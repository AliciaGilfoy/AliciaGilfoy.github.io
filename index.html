CMA-Tracker

This is a full-stack live program with over 45 registered users. During the Covid-19 Pandemic I saw a need for a point tracking and spending program for a small taekwondo studio. I approached the owner of the studio and coded this app. It allows users to register their children and submit and track daily activities for points. They can then spend their points in a store. There is security set up that tags certain users as admins or instructors. A student may not submit points for the same day twice. This was build with a vue front-end and a node back-end.
